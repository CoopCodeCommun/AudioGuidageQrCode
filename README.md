# Projet Audio guidage par qrcode:

## Instalation :

### Installez les dépendances NPM

- A la racine

```
npm i
```

- dans "front"

```
npm i
```

### Build pour le web

```
npm run build
```

### Build pour application mobile

```
cordova build android
cordova run android
```

### Debug sur navigateur - "chrome://inspect/#devices"
